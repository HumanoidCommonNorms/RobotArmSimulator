# スクリプト

このディレクトリには、以下のスクリプトが含まれています。
作業を自動化するために使用できます。


## 00_setup_venv.cmd

仮想環境をセットアップするスクリプトです。
venvを作成し、activateします。
venv名は「venv_VRChatSyncPy」としてます。

## 01_install.cmd

必要なパッケージをインストールするスクリプトです。
また、VRChatSyncPyをインストールします。

## 02_run_sample.cmd

アプリケーションを実行するサンプルスクリプトです。


## 03_create_exe.cmd

アプリケーションを実行可能なEXEファイルに変換するスクリプトです。
